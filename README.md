# MeowFit 🐱 (猫猫要健康！)

> “喵呜~ 想要拥有完美身材吗？让本喵来监督你的每一餐和每一次运动吧！”

**MeowFit** 是一款基于 Android 的个人健康管理应用，也是你最贴心的健康小管家。它不仅能帮你追踪每日的热量摄入与消耗，还能通过可视化的方式管理体重目标。无论是减肥、增肌还是保持健康，MeowFit 都会陪你一起努力，直到你变成最完美的自己！✨

---

## 🐾 功能亮点 (Features)

### 1. 📝 铲屎官的专属档案 (Profile & Goals)
- **定制化目标**: 无论你是想变瘦、变壮还是保持现状，告诉本喵你的身高、体重和年龄，我会为你计算出专属的 **每日基础代谢 (BMR)** 和 **总能量消耗 (TDEE)**。
- **灵活调整**: 随时更新你的体重，我会帮你调整计划，确保你走在正确的道路上！

### 2. 📊 今日喵运势 (Dashboard)
- **实时监控**: 一眼看清今天的摄入、消耗和剩余热量额度。
- **记录生活**: 吃了什么？做了什么运动？统统记下来！支持点击修改和删除，方便极了。
- **体重打卡**: 每天记录体重，看着曲线慢慢变好，心情也会变好哦~
- **运动计时**: 准备开始运动了吗？打开计时器，我会陪你一起流汗！💦

### 3. 🤖 AI 喵助手 (Smart AI Recording)
- **智能对话**: 懒得查热量？直接告诉我“我吃了一个苹果”，本喵会动用超强 AI 大脑帮你自动分析热量并生成记录！
- **多模型支持**: 支持 OpenAI, DeepSeek, Moonshot, Aliyun 等多种 AI “大脑”，任你挑选。

### 4. 📈 喵喵数据中心 (Statistics)
- **多维度分析**: 按周、月、年查看你的运动和饮食数据，不放过任何一个细节。
- **体重趋势图**: 清晰的折线图，让你直观感受到身体的变化。

### 5. 📅 爪印日历 (Heatmap Calendar)
- **热力图**: 像 GitHub 的绿格子一样，用颜色展示你每天的热量盈余/亏缺情况。让每一天都充满色彩！
- **历史回顾**: 点击日历，随时查看过去某一天的详细记录。

### 6. 🔒 数据安全 (Data Privacy)
- **本地存储**: 你的所有秘密都安全地保存在本地 SQLite 数据库中，本喵绝不偷看！
- **备份与恢复**: 怕数据丢了？一键导出 JSON 备份，随时恢复，安全感满满。

## 🛠️ 技术栈 (Tech Stack)

本项目是用满满的爱意 ❤️ 和以下技术构建的：

- **语言**: Kotlin 🐈
- **UI**: Jetpack Compose (Material Design 3) 🎨
- **架构**: MVVM (Model-View-ViewModel) 🏗️
- **数据库**: Room (SQLite) 🗄️
- **网络**: Retrofit (用于 AI 功能) 🌐
- **异步处理**: Coroutines & Flow 🌊
- **导航**: Jetpack Navigation Compose 🗺️

## 🚀 如何领养 (How to Run)

1.  使用 **Android Studio** (推荐 Giraffe 或更新版本) 打开 `MeowFit` 目录。
2.  等待 Gradle 同步完成 (就像等待猫粮倒满一样耐心~)。
3.  连接 Android 设备或启动模拟器 (最低支持 Android 8.0 / API 26)。
4.  点击 "Run" (▶️) 按钮，唤醒你的 MeowFit！
5.  (可选) 在设置中配置 AI API Key，解锁超强智能喵助手功能。
